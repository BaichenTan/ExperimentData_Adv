This repo contains data for the publicly available datasets listed below.
Each subfolder also contains papers and readme files (if any) associated with that dataset.
The sample R file gettingDataIntoR.R shows how the data can be read into R.


==========
==========

Datasets:

dataset_trimmed_extraversion_final.csv: trimmed dataset that we used for causal effect estimatioin. Have ordered the data units based on the raw extraversion score from low to high.

dataset_trimmed_agreeableness_final.csv：trimmed dataset that we used for placebo analysis. Have ordered the data units based on the raw agreeableness score from low to high.

==========
==========

Placebo_Effect_Agreeableness.ipynb：code that checks the placebo variable "agreeableness"
